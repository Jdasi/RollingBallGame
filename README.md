# Overview

A simple platforming prototype made in Unreal 5.7.3, primarily written in C++ and supplemented with Blueprints for level elements. The player controls a ball that can jump and launch forward using powerups. The prototype is fully playable and lasts around 5 minutes.

# Features

- Fully C++ implemented player pawn with responsive ground and air movement
- Generic powerup base Blueprint class with overridable on-pickup behavior
- event-driven UI
- Dynamic mesh creation blueprint for wires
- Checkpoint system
- Audio with continuous modulation controlled by C++
- Generic power receiver system with overridable on-power-lost behavior
- World subsystem to control slo-mo
- Player ability persistence via custom PlayerState
